# 🧙 Izzy Wizzy — Менеджер персонажей D&D 5e

Локальное приложение для ведения листов персонажей Dungeons & Dragons 5-й редакции на русском языке.  
Работает полностью офлайн, не требует интернета и регистрации.

---

## ✨ Возможности

- 📋 Полный лист персонажа — характеристики, навыки, спасброски, атаки, заклинания
- 🎲 Броски кубиков с историей и отправкой в Discord
- 📤 Экспорт/импорт в формат **LongStoryShort** (перевод dnd.su)
- 📄 Экспорт заполненного **PDF** (официальный лист персонажа D&D 5e)
- 💾 Автосохранение — все данные хранятся локально на вашем компьютере
- 🖼️ Портреты персонажей с кадрированием
- ⚗️ Мастер создания персонажа


---

## 🚀 Быстрый старт

### Вариант 1 — Готовый `.exe` (рекомендуется)

1. Скачайте `IzzyWizzy.exe` со страницы [Releases](../../releases/latest)
2. Запустите файл — браузер откроется автоматически
3. Готово! Персонажи сохраняются в папке `characters/` рядом с `.exe`

> ⚠️ Windows может показать предупреждение SmartScreen — нажмите «Подробнее» → «Выполнить в любом случае»

---

### Вариант 2 — Запуск из исходников

**Требования:** Python 3.10+

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/your-username/izzy-wizzy.git
cd izzy-wizzy

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите
python app.py
```

Откройте браузер: [http://localhost:5000](http://localhost:5000)

---

## 🔨 Сборка `.exe` самостоятельно

```bash
pip install pyinstaller
pyinstaller izzy_wizzy.spec
```

Готовый файл появится в папке `dist/IzzyWizzy.exe`

## 📁 Структура файлов

```
izzy-wizzy/
├── app.py                  # Flask-сервер
├── fill_pdf.py             # Генерация PDF
├── izzy_wizzy.spec         # Конфиг PyInstaller
├── requirements.txt
├── templates/
│   └── index.html
├── static/
│   ├── js/app.js
│   ├── data/               # Расы, классы, заклинания...
│   ├── img/
│   ├── DejaVuSans.ttf
│   └── sheet_template.pdf
└── characters/             # Сохранённые персонажи (создаётся автоматически)
```
